Groove Media Audio Converter 1.1.1
By Anthony Kleine

	This command line tool converts the Groove Media Audio (GMA) codec used by 3D Groove GX to PCM.
	
	Groove Media Audio is a proprietary audio codec by OTOY with 2x better compression than MP3. The codec may be used in WAVE Audio Files.

	Requires the Groove Media Audio Extension (GrooveMediaAudio.dll).

Usage: GrooveMediaAudioConverter runaway.WAV runaway_converted.WAV
